Discord Tweaks (gist)

A collection of small CSS tweaks for Discord. This requires browser with CSS injection extension or 3rd party client.

It's personalized for my liking, so modify and remove feature by editing it yourself!

Overview:

  • Remove user customization. i.e. custom font and nameplate background.
  • Remove border curve in most element.
  • Remove unnecessary buttons like help (press F1), gift, discover, and add app.
  • Shrink chat bar to eat less space.
  • Change "DM" button icon into Illith! Go to @shshaw codepen to make your own svg.
  • Other visual changes.

Youtube CSS (gist)

Mostly tweaks for description and comment view. Designed for dark theme but light theme has preset color in the css file. Use CSS inject extension or YouTube Enhancer (not sponsored) to use.

The background (and associated comment color) is disabled by default as it uses Steam for the wallpaper.

This also remove voice search button, just because.

Old Reddit CSS (gist)

A three in one hacks for old Reddit. It meant to be used with Reddit Enhancement Suite (RES) extension.

Hide spoiler by prefixing [SPOILER] and show first few character of the title. Expanded when hovered but also less visible with bad colors to make sure you don't accidentally see it when accidentally hover it. Doesn't work on search result.

Pin OP comment inside a thread. This has priority over admin pin. It modify comment display into flex so with some site-specific CSS it has potential to break!

Final mod is to make sure up and down vote button always shows. Not really useful if you disable site-specific CSS but it's there.
