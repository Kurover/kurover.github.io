Start Orb - Illith

This is a start orb replacement for Open-Shell Menu but more generic version is also available if you use other mod. To use, save the .png below to a safe directory. Then replace the orb menu in the settings.

Btw, if you can code/fork Open-Shell to enable animation other than Normal to Hot to pretty much all other combination (Normal to Pressed, vice versa, Hot to Pressed, vice versa. Refer to this document page), I'll trade you a start menu animation with character of your choice. Contact me if you're interested, yea? I have sample animation of Normal x Pressed, but of course the program isn't allowed to render it.
