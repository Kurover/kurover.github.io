Start Orb - Illith

This is a start orb replacement, mainly for Open-Shell Menu. It has an extended animation when hovered. To use, save the .png below to a safe directory. Then replace the orb menu in the settings.

If you use other start menu mod or replacement, grab the generic copy instead.

Btw, if you can code/fork Open-Shell to enable animation other than Normal to Hot to pretty much all other combination (Normal to Pressed, vice versa, Hot to Pressed, vice versa. Refer to this document page), I'll trade you a start menu animation with character of your choice. Contact me if you're interested, yea? I have sample animation of Normal x Pressed, but of course it can't render it.
