# Choose Yo Fonts!

I've been trying out some other fonts for the website, mostly the mono space kind. So many choice but I ended up only picking 1, Fira Code, which also replace any current page that use Consolas. I swapped them out because Consolas isn't exactly "free" to use per se. It's prepackaged in Windows but yeah.

I didn't pick pixel-arty font because imo they're terrible for long read. Add the dark mode on top and you have recipe for disaster on anyone with astigmatism (I had to look up this word spelling). It's a bit better regardless with the color update, which is the main reason why I did it, but just because I tipped the balance once, doesn't mean I can tip it the other direction just because. I do however found one I liked, [BigBlue Terminal](https://int10h.org/blog/2015/12/bigblue-terminal-oldschool-fixed-width-font/). [Nerdfont](https://www.nerdfonts.com/font-downloads) expanded it more if you like it too.
