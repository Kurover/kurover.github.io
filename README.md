# Website
A general purpose website handcrafted purely with HTML and CSS to channel the spirit of pixel-art limitation in technique and presentation. Aimed to be performant, pocket-sized, and modestly styled; with mobile layout and spectacular keyboard-nav support (until you go to File page...)!

Jekyll for automation and the site use 99% vanilla CSS with the rest of 1% dedicated to "double slash to comment" in SCSS because `/**/` is, affectionately, cringe unlike AI art that is fully **cringe**.

:warning: Currently under development! [Old Version](https://kurover.github.io/v2) is up and running in frozen state.
